# Análisis Espacial de Riesgo Sísmico en la Ciudad de México

## Descripción

Este proyecto tiene como objetivo desarrollar un flujo de trabajo para el análisis espacial del riesgo sísmico en la Ciudad de México mediante herramientas SIG y técnicas de visión por computadora.
Se integran datos geoespaciales, procesamiento de imágenes satelitales y modelos de aprendizaje profundo para identificar edificios y generar información que apoye la evaluación de vulnerabilidad urbana.

---

## Objetivos

- Integrar información geoespacial de diferentes fuentes.
- Limpiar y preparar escenarios de análisis.
- Procesar imágenes satelitales.
- Detectar edificios utilizando modelos basados en YOLO.
- Generar insumos para futuros análisis de vulnerabilidad y riesgo sísmico.

---

## Herramientas utilizadas
